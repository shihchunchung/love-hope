# 新版分類內容對應

日期：2026-05-19

本文件承接 `docs/original-site/` 的舊站事實盤點，將舊站內容對應到新版 `index.html` 已確定的分類。此文件不重新討論分類命名。

## 1. 前提

新版主分類以目前 `index.html` 導覽為準：

- 關於我們
- 慈善身影
- 公益行動
- 真愛多多
- 最新消息
- 文創小舖
- 搖籃咖啡
- 場地租借

第一階段目標是讓首頁與核心分類內容回到真實舊站內容，不再依賴示意文案。

- 文創小舖是否有真實商品資料、照片與購買方式。
- 搖籃咖啡是否直接外連 `https://www.facebook.com/LoveHopeYLC/`。
- 捐款方式是否仍使用舊站銀行帳號、劃撥帳號、信用卡授權書。
- 捐款與徵信要放在哪個導覽下，或用全站 CTA + footer 入口。
- 董事名單是否仍為最新版。
- 場地租借 PDF 2014/9/24 版本是否仍有效。
- 最新消息第一階段要顯示幾筆，以及是否只顯示 2024 公告與代表性文章。
- 「熱門閱讀」是否符合基金會網站語氣，或應改為「重點故事」。

## 8. 實作順序建議

1. 先修 `index.html` 的內容真實性：移除示意日期、示意數字、虛構熱門排行。
2. 首頁保留分類導覽，但把區塊改成 P0 內容摘要。
3. 建立 P0 內頁骨架：`about.html`、`people.html`、`actions.html`、`true-love.html`、`news.html`、`venue.html`。
4. 把捐款資訊做成全站 CTA 或 `donate.html`。
5. 再補文創小舖、搖籃咖啡與新聞歸檔。
