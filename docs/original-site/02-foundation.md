# 02 認識薛伯輝基金會

## 2.1 緣起

URL：`https://www.lovehope.org.tw/緣起.html`

### 頁面角色

基金會正式介紹頁。與 `董事.html` 互為子導覽。

### 主要文字

- 2004 年，薛伯輝先生與陳惠如女士一同出席國際扶輪社在日本大阪市召開的年度大會。前往和歌山縣那智勝浦瀑布遊覽區遊玩，因一場意外車禍，薛伯輝先生離開親人與朋友。
- 陳惠如女士與孩子們為紀念薛伯輝先生，於 2008 年將「財團法人薛氏文教基金會」更名為「財團法人薛伯輝基金會」。
- 基金會所在地選在薛伯輝先生事業起點，播種希望、散播愛。

### 基金會簡史

| 日期 | 事件 |
|---|---|
| 1975-01-30 | 「財團法人薛氏文教基金會」成立，以提倡文化教育、發揚倫理孝道為宗旨，隸屬台北市教育局。 |
| 2008-08-22 | 陳惠如女士為紀念先生，將「財團法人薛氏文教基金會」更名為「財團法人薛伯輝基金會」。 |
| 2013-11-07 | 為將愛傳遞得更遠、更廣，基金會從「台北市教育局」轉隸於「教育部」。 |

### 基金會宗旨

提倡文化教育，發揚倫理孝道，依法令規定辦理：

- 捐贈文化教育之圖書。
- 設立獎學金獎勵優秀學生。
- 舉辦文化教育有關活動。
- 捐贈興建文化教育之相關設施或機構。
- 捐贈或辦理特殊教育活動經費。
