# 03 關於薛伯輝先生

URL：`https://www.lovehope.org.tw/關於薛伯輝先生.html`

## 頁面角色

基金會命名人物與精神來源介紹頁。

## 主要內容

- 薛伯輝先生，名永華，1929 年生於江蘇省海門縣三陽鎮。
- 1949 年自上海來台，協助父親薛頌留先生創辦「大中國圖書公司」。
- 曾任新光標準內衣染織整理廠股份有限公司總經理。
- 1964 年創辦國華化學工業股份有限公司，生產尼龍絲，為台灣第一條尼龍生產線。
- 後成立中華化學纖維股份有限公司，並於 1970 年代擴展至泰國與中國等地。
- 與新光證券、美亞鋼管、國華碳纖維、國際票券金融、聯成石油化學、中國建築經理、世界先進積體電路等公司創立高度相關。
- 舊站以「紳士企業家」描述其誠實、嚴謹、親和幽默、樂觀開朗、熱愛閱讀、不菸不酒。
- 長期支持父親創立的薛氏文教基金會，幫助清寒子弟，並關注家鄉海門建設。
- 與陳惠如女士於 1949 年結婚，共同撫育三名兒女。
- 1986 年加入仁愛扶輪社後與夫人投入公益。
- 2004 年於日本參加國際扶輪社年度大會期間不幸因故喪生。
- 陳惠如女士為延續其精神與遺志創立薛伯輝基金會。
