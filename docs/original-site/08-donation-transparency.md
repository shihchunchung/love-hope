# 08 捐款與徵信

## 8.1 捐款方式

URL：`https://www.lovehope.org.tw/捐款方式.html`

### 頁面角色

捐款方式說明頁，並作為捐款徵信與表單下載入口。

### 銀行匯款 / ATM 轉帳

- 銀行：國泰世華 台北分行
- 銀行代碼：013
- 帳號：007-03-603896-3
- 戶名：財團法人薛伯輝基金會

### 郵局劃撥

- 帳號：50067771
- 戶名：財團法人薛伯輝基金會
- 頁面有「郵政劃撥單參照寫法」連結，檔案待補查。

### 信用卡捐款

- 下載並填寫「信用卡授權書」。
- 以傳真或郵寄方式交回基金會。
- 授權書檔案待補查。

### 支票捐款

- 支票抬頭：財團法人薛伯輝基金會。
- 填寫金額與抬頭後，掛號寄至基金會。

### 現場捐款

- 可親自前往基金會。
