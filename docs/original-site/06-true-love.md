## 6.2 伯利恆早療暨融合教育中心

URL：`https://www.lovehope.org.tw/伯利恆早療既融合教育中心.html`

### 頁面角色

真愛多多核心專案頁。說明甘惠忠神父心願與薛伯輝基金會使命。

### 主要內容

- 主題：興建「伯利恆早療暨融合教育中心」。
- 舊站說明 0 至 3 歲為早療黃金時期。
- 甘神父期盼在台南市學甲區華宗路上、慈母幼兒園後方興建中心。
- 中心預計滿足時段療育、專業教室、日間托育等需求。
- 期盼推動南台灣正確融合教育觀念、重視師資培訓，成為政府與民間橋樑。
- 中心除專業早療教室與融合教室外，另設「愛的圖書館」。
- 愛的圖書館分布於 2、3 樓，約 275 平方公尺。
- 圖書館以生活出發，透過小廚房活動課程進行孩童五感教育。
- 邀請專家每週提供家長個別諮詢。
- 結合社區，成為社區關懷據點，舉辦親職講座、融合教育講座，並成立家長與老師交流平台。

| 項目 | 內容 |
