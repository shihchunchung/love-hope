# 舊網站逐頁盤點

盤點日期：2026-05-19

本目錄記錄 `https://www.lovehope.org.tw/` 舊站內容事實。目標是支撐後續純 HTML 重構：先知道舊站完整內容，再決定新版分類、首頁區塊與內頁順序。

## 原則

- 先記錄舊站事實，不做新版分類取捨。
- 每個入口頁都要追查子頁，不能只記主導覽。
- 每頁都要記錄 URL、頁面角色、主標題、副標、主要文字、站內連結、外部連結、圖片資產、PDF / 表單與待確認事項。
- 新聞與活動屬列表型內容，先完整記錄列表入口、主題群與代表頁；逐篇文章全文搬移應在「內容搬遷」階段另列批次處理。
- 舊站大量使用 table 排版、圖片切圖、image map、iframe 與 PDF；這些都要在文件中保留來源線索。

## 待補查清單

- 文創小品是否有頁面或只是未連結文字。
- 加入會員是否有頁面或表單。
- 相關連結頁內容與外部連結狀態。
- 全部新聞 / 活動文章逐篇全文是否都要進新版，或只保留索引與重要文章。
- 捐款徵信 PDF 的完整年度清單。
- 信用卡授權書與郵政劃撥單參照寫法檔案。
